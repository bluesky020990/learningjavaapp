Install boostrap for Angular

Install Boostrap

Ngồi nhìn cái giao diện nó xấu quá, định dùng Material nhưng có vẻ ko đẹp bằng nên quyết định đi install boostrap vào cho đẹp

Lesson này chỉ là chỉ cách cài đặt và sử dụng Boostrap mà thôi.

Nào, cùng tiến hành thôi.

Bước 1, install ng-boostrap

npm install ngx-bootstrap bootstrap --save

Bước 2, add boostrap vào template

Mở file angular.json và add thêm đoạn này vào

"styles": [
    "./node_modules/@angular/material/prebuilt-themes/deeppurple-amber.css",
    "./node_modules/bootstrap/dist/css/bootstrap.min.css",
"src/styles.scss"
],
