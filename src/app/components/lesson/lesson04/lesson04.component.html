Bài 4. Angular và Material

Trong bài học này ta tiến hành nhúng cái material vào cho angular cho đẹp.

Material có những thư viện khá thú vị. Việc sử dụng các module này sẽ giúp ta tiết kiệm phần nào thời gian.

Không dài dòng nữa, đi vào bài học thôi.

 1. Install Angular Material

Để tiến hành cài đặt full Material cho Angular ta cần install Angular Material trước. Cách làm như sau :

npm install --save @angular/material @angular/cdk @angular/animations

Sau đó, ta tiến hành cài đặt Material cho Angular. Thực chất step này để cài theme, thêm các thư viện về animation.

ng add @angular/material

Install thêm cái Hammer nữa

npm install --save hammerjs

Add thêm stylesheet cho Material Icon nữa.

Xem thêm thông tin chi tiết tại link này material.angular.io

 2. Sử dụng các module trong Material như thế nào?

Có nhiều cách để import các module của Material vào, nhưng ta sẽ đi theo cách mà rõ ràng và đơn giản nhất.

1. Tạo Material Module
ng g m Material

Tiếp đó, vào module này, tiến hành import các Module mà ta cần xài. Cái này gồm có 2 phần, import và export.

Sau đó, quay lại trang app.module.ts, add cái Module này vào thôi.
